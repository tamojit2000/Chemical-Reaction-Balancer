# Chemical Reaction Balancer
 Balances any chemical reaction

### Modules Used

- PyQt5 (UI)
- tkinter (UI)
- numpy (Calculation Matrix)
- os

### Features

- Balances any equation.
- Previous solved equations stored for quick access.
